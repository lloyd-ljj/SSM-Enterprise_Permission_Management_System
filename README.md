# SSM-Enterprise_Permission_Management_System
这是一个企业权限管理系统，主要包括用户管理、角色管理、资源权限管理、权限关联控制、产品管理、订单管理、日志查询等功能。

## 环境

- Java版本：1.8
- 数据库：Oracle 10g
- 服务器：Tomcat
- 开发工具：IDEA 2019
- 版本管理工具：Maven

## 项目技术架构(Spring + SpringMVC + Mybatis)

- Maven
- Spring
- SpringMVC
- Spring Security（登录认证、账户权限管理）
- MyBatis
- Bootstrap
